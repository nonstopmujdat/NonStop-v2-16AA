# NONSTOP V1.9 Notları

Bu sürüm artık sadece HTML prototip değil, Next.js içinde çalışabilecek gerçek MVP kod yapısıdır.

## Çalışan kısımlar

- Operatör ekranı
- Skor güncelleme
- Süre başlat / durdur
- Oyuncu seçimi
- Sahadakiler / yedekler
- Oyuncu değişikliği popup
- Basket sonrası Asist / Faul / YOK popup
- Serbest atış
- Ribaund, top çalma, blok, top kaybı, faul kayıtları
- Offline queue localStorage
- Online/offline simülasyonu
- API route örnekleri

## Gerçek sisteme bağlanması gerekenler

- Supabase Auth
- Supabase insert işlemleri
- Gerçek match_id, player_id, team_id verileri
- PDF rapor üretimi
- WebSocket / realtime senkronizasyon
- RLS yetkileri
