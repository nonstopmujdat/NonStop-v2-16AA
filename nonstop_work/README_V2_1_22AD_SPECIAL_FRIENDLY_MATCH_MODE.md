# NONSTOP V2.1.22AD - Special / Friendly Match Mode

Bu sürüm özel maç ve hazırlık maçı akışını ekler.

## Amaç
- U14 takımının U16 takımıyla maç yapabilmesi
- Karma takımla maç yapılabilmesi
- Hazırlık / özel maçların lig puan durumuna karışmaması
- Maç içi oyuncu istatistiği, takım istatistiği, faul/şut haritası tutulması

## Kurallar
- SPECIAL_MATCH ve FRIENDLY maçları varsayılan olarak puan durumuna işlemez.
- Supervisor isterse sezon istatistiğine işlenebilir seçeneği kullanılabilir.
- Operatör sayfasında özel maç oluşturma demo akışı vardır.
